FAQ

Questions? We've got answers.

Does Spravio work with any Jira plan?

Yes — Free, Standard, Premium and Enterprise. We use Jira's public REST API so nothing changes on your end.

Does Spravio support Azure DevOps?

Fully. Work items, boards, repositories and pull requests — same feature parity as our Jira + GitHub pipeline.

How does the GitHub integration work?

Install the Spravio GitHub App on the orgs and repos you want to sync. We read PRs, commits, and review activity — no write access.

What is a Project Manager (GP) role?

In agencies, a GP (Gerente de Projeto) owns one or more client engagements. Spravio scopes dashboards per GP, so they only see what they're responsible for — while owners see everything.

Is my data secure?

Data is encrypted in transit and at rest. We store the minimum needed for our metrics. SSO and SCIM available on Scale.

Can I cancel anytime?

Yes. Monthly plans cancel instantly. Annual plans renew yearly — cancel before renewal and you won't be charged.

Do you offer a free trial?

14 days on every plan. No credit card required to start.
